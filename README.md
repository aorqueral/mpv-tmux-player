# mpv-tmux-player

Reproductor de música en terminal basado en mpv + tmux + fzf.

## Features
- mpv daemon controlado por socket
- búsqueda interactiva con fzf
- dashboard ASCII con cover (chafa)
- integración con i3wm (keybinds)
- local + YouTube

## Requisitos
- mpv
- yt-dlp
- fzf
- chafa
- tmux

## Uso
- mp play <query>
- mp search <query>
- mp pause
